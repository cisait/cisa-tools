Web Outlook on the Web (outlook.office.com)
  1. Click Copy Signature above
  2. Go to outlook.office.com and sign in
  3. Click the gear icon (Settings) in the top-right corner
  4. Search for "Email signature" in the settings search bar and select it
  5. Click + New signature, give it a name, then paste into the text box
  6. Under Select default signatures, set it for new messages and replies
  7. Click Save
Desktop Outlook Desktop App (Windows)
  1. Click Copy Signature above
  2. Open the Outlook desktop app
  3. Go to File → Options → Mail
  4. Click the Signatures… button
  5. Click New, name your signature, then paste into the editor
  6. Under Choose default signature, set it for new messages and replies/forwards
  7. Click OK twice to save
